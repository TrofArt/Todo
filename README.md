## Начало работы

Данное руководство поможет вам получить копию проекта и запустить его на вашем локальном компьютере для разработки и тестирования.


## Требования

- Операционная система: Windows 10 или новее
- Visual Studio 2022 с установленным пакетом для разработки приложений на .NET (версии 8.0)

- 
## Установка
- Склонируйте репозиторий с помощью интерфейса Visual Studio.
- Откройте проект и переключитесь на нужную ветку (например, Lab1).
- Запустите проект через меню или нажатием клавиши F5.
- При необходимости внесите изменения в файл MainWindow.xaml.cs, чтобы настроить стартовую форму.
